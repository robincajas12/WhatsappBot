*🔒 Comandos de Administrador (Solo Tú):*
• `!scripts`: Lista los scripts en la carpeta `scripts/`.
• `!run <script> [args]`: Ejecuta un script.
• `!busy on` / `!busy off`: Activa/desactiva el modo ocupado.
• `!add <número>` / `!remove <número>`: Permite/bloquea la IA para un usuario.
• `!recordar <tiempo> <mensaje>`: Guarda un recordatorio persistente (ej: `!recordar 3d comprar agua`).
• `!config set sleep <inicio> <fin>`: Cambia las horas de descanso (ej: `!config set sleep 21 5`).
• `!save [as <categoría>]`: (Cita un mensaje) Guarda el mensaje de texto citado.
• `!save file [as <categoría>]`: (Cita un archivo/imagen) Descarga y guarda el archivo en el servidor.
• `!saved`: Lista todos los elementos guardados con sus IDs correspondientes.
• `!saved <ID>`: Envía el mensaje de texto o el archivo descargable asociado a ese ID.
• `!saved <tel>`: Filtra los elementos guardados según el número de contacto.
• `!saved key <categoría>`: Filtra los elementos guardados según la categoría.
• `!saved delete id <ID>`: Elimina permanentemente el elemento asociado a ese ID.
• `!saved delete <tel>`: Elimina todos los elementos guardados de un contacto.
• `!saved delete key <categoría>`: Elimina todos los elementos guardados en una categoría.
• `!enlace [<nombre> [<url>] / delete <nombre>]`: Guarda, consulta o elimina enlaces importantes.
• `!archivo [<nombre_archivo>]`: Lista archivos del servidor o descarga uno.
